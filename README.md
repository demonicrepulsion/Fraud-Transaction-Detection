# Fraud Transaction Detection System

Machine Learning project for detecting fraudulent transactions using classification and anomaly detection techniques.

## Features
- Data Preprocessing
- Feature Scaling
- SMOTE Balancing
- Logistic Regression
- Random Forest
- XGBoost
- Explainable AI (SHAP)
- Real-Time Prediction Simulation

## Technologies
- Python
- Pandas
- NumPy
- Scikit-Learn
- XGBoost
